Rosbags are meant to help you accomplish Task 1.

HOW TO:
    1- Change directory to where you haved saved your rosbags.
    2- Run the rosbag(s) --> $rosbag play sample#.bap OR $rosbag play --pause sample#.bag
    2- You can now view  all the published topics --> $rostopic list
    3- You can also view the actual robot moving while recording the bag--> $rosrun image_view image_view image:=//
    camera/rgb/image_color
    3- Refer to the README file in helperscripts folder to guide you on how to accomplish the rest of Task 1.
